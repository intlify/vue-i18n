# 公式ツール

## Vue CLI プラグイン

[vue-cli-plugin-i18n](https://github.com/kazupon/vue-cli-plugin-i18n) は、Vue CLI プラグインとして公式に提供されています。

このプラグインを使用すると、Vue アプリケーションの i18n 環境をセットアップし、i18n 開発環境をサポートできます。

## ESLint プラグイン

[eslint-plugin-vue-i18n](https://github.com/intlify/eslint-plugin-vue-i18n) は、Vue I18n 用の ESLint プラグインです。

いくつかのローカリゼーション lint 機能を Vue.js アプリケーションに簡単に統合できます。

## Intlify CLI

[@intlify/cli](https://github.com/intlify/cli) は、i18n 開発用の CLI ツールです。

`intlify compile` コマンドを使用して、i18n リソース（`json5?`、`ya?ml`）をプリコンパイルできます。

## 拡張機能

[vue-i18n-extensions](https://github.com/intlify/vue-i18n-extensions) は、Vue I18n のいくつかの拡張機能を提供します。

この拡張機能を使用して、SSR を有効にし、i18n のパフォーマンスを向上させることができます。

## Vue 2.x 用 Composition API

[vue-i18n-composable](https://github.com/intlify/vue-i18n-composable) は、Vue 2.x の Vue I18n 用の Composition API を提供します。

## Petite-Vue-I18n

[petite-vue-i18n](https://github.com/intlify/vue-i18n/tree/master/packages/petite-vue-i18n) は、最小限の機能のみを提供する Vue I18n の代替ディストリビューションです。

petite-vue-i18n はまだ実験段階であるため、バグやサポートされていないユースケースが発生する可能性があることに注意してください。
