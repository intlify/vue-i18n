# 官方工具

## Vue CLI 插件

[vue-cli-plugin-i18n](https://github.com/kazupon/vue-cli-plugin-i18n) 作为 Vue CLI 插件官方提供。

使用此插件，你可以为 Vue 应用程序设置 i18n 环境，并支持 i18n 开发环境。

## ESLint 插件

[eslint-plugin-vue-i18n](https://github.com/intlify/eslint-plugin-vue-i18n) 是 Vue I18n 的 ESLint 插件。

它可以轻松地将一些本地化 lint 功能集成到你的 Vue.js 应用程序中。

## Intlify CLI

[@intlify/cli](https://github.com/intlify/cli) 是用于 i18n 开发的 CLI 工具。

你可以使用 `intlify compile` 命令预编译 i18n 资源（`json5?`, `ya?ml`）。

## 扩展

[vue-i18n-extensions](https://github.com/intlify/vue-i18n-extensions) 为 Vue I18n 提供了一些扩展。

你可以使用此扩展来启用 SSR 并提高 i18n 性能。

## Vue 2.x 的组合式 API

[vue-i18n-composable](https://github.com/intlify/vue-i18n-composable) 为 Vue 2.x 中的 Vue I18n 提供了组合式 API。

## Petite-Vue-I18n

[petite-vue-i18n](https://github.com/intlify/vue-i18n/tree/master/packages/petite-vue-i18n) 是 Vue I18n 的替代发行版，它仅提供最小的功能。

请注意，petite-vue-i18n 仍处于实验阶段，因此你可能会遇到错误和不支持的用例。
