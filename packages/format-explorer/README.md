# @intlify/message-format-explorer

Intlify message format explorer

## 🔨 develop the format explorer

```sh
$ yarn dev:explorer
```

## 📦 build the format explorer

```sh
$ yarn build:explorer
```

## 🏄 serve the format explorer

```sh
$ yarn serve:explorer
```
